*** WICHTIG ***
Bevor man mit einer MySQL-Datenbank arbeiten kann, 
muss man ggf. erst den Connector zu den Libraries hinzufuegen!

Project -> Properties -> Java Build Path -> Libraries -> Add External JARs
- dann den MySQL-Connector suchen:
  mysql-connector-java-5.0.8-bin.jar

hinzufuegen, OK klicken.
